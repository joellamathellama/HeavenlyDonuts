# Mock Website for a fake donut store

Status: Fresh app, currently testing things out.

### Tech stack
- React
- Redux
- Gulp
- Gulp-sass
- Gulp-connect
- Gulp-uglify
- Express
- Browserify
- Babelify
- SQLite3
- Knex

<hr/>
##### Development

$ npm install

$ knex migrate:latest

$ knex seed:run

$ npm run dev

$ visit localhost:8000
